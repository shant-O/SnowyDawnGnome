❄️SnowyDawn is a Touchscreen Friendly GTK Light theme featuring rounded and smooth geometry with subtle colors🎨

🎭The 90% opacity light mode backgrounds are friendly to the eyes that won't make you feel like you're looking at the SUN!🌝
🎭This theme has contrasted gradient effects to make the light mode even more stylish.

📌Dark Version of this theme "Cyber Dusk - Rounded Glass" is available at:
🔗https://www.gnome-look.org/p/2357650

🛠️Installation:
🔹1. Extract the archive.
🔹2. Enter the SnowyDawn Folder and FIRST thing: 
	MOVE (🚫do not copy) the gtk-3.0 and gtk-4.0 folders into
	~/.config/ (if you have pre-existing gtk folders here,
	back them up before)
🔹3. Now "gnome-shell" should be the only FOLDER inside the
	SnowyDawn folder. Copy the SnowyDawn folder to either
	~/.themes/ or ~/.local/share/themes/
🔹4. If you don't have "User Themes"🧩 gnome extension installed already,
	install it and then choose the theme from its menu and done!
📌[ on some version of gnome, mostly after gnome 49.0,
having any ⚠️gtk-* folder inside the themes folder causes the gnome
desktop to freeze⚠️, so its best practice to MOVE them beforehand]
